## Install

```bash
sail up -d
sail npm run dev
```
Добавить пользователя
```bash
sail artisan users:create {name} {email} {pass}
```
Оперции с балансом пользователя:
```bash
sail artisan wallet:operation
--Следовать инструкциям
```
